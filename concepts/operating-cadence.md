# Operating Cadence

Operating cadence is the rhythm that turns project management from episodic rescue into a control system. It defines when decisions happen, when work is reviewed, when risks are surfaced, and when plans are updated.

In weak organizations, cadence is replaced by interruptions. In strong ones, cadence absorbs most coordination so people can spend more time executing.

## Why it matters more in the age of AI

Agentic systems increase output volume. That sounds good until you realize it also increases:
- decisions to make
- artifacts to review
- state changes to track
- ways for work to silently drift

Without cadence, AI-assisted execution creates backlog fog: lots of motion, weak control.

## Core idea

Cadence should do four things:
1. synchronize understanding
2. expose blockers and risks early
3. force prioritization under constraint
4. create learning loops from what actually happened

## Typical layers

### Daily
- blocked/unblocked updates
- handoff clarification
- review queue management
- urgent risk surfacing

### Weekly
- backlog reprioritization
- dependency review
- quality/telemetry review
- staffing or agent allocation changes
- decision log cleanup

### Milestone / phase
- scope validation
- acceptance review
- retrospective / postmortem
- operating-model adjustments

## AI-native twist

In AI-heavy teams, the most important weekly question is not "what got done?" It is:
- what work completed cleanly without intervention?
- what required heavy human rescue?
- what failure modes repeated?
- where is review capacity now the bottleneck?

That is how cadence becomes a scaling mechanism instead of a meeting ritual.

## Failure modes
- too many meetings, too little truth
- no stable review rhythm, so approvals become random
- telemetry collected but never used
- escalation only after visible failure
- daily standups that track activity instead of blocked flow

## Practical rule

Run the minimum cadence that keeps the system legible. If a meeting does not change prioritization, unblock work, or improve control, it is probably status theater.
