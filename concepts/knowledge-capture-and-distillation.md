# Knowledge Capture and Distillation

Execution without distillation is organizational amnesia.

If every good run dies in a chat log, the system does not improve.

## Distillation targets
Turn repeated work into:
- checklists
- templates
- issue shapes
- prompts
- skills
- automation
- guardrails
- dashboards

## What to capture
- successful procedures
- recurring failure modes
- environment quirks
- decision rationales
- high-value examples
- review heuristics

## Distillation pipeline
1. Notice repetition
2. Extract the pattern
3. Compress it into a reusable artifact
4. Reuse it in future execution
5. Patch it when reality changes

## Good artifacts
A good artifact is:
- short enough to use
- specific enough to guide action
- stable enough to matter later
- easy to revise when wrong

## The compounding effect
The organizations that win with AI are not the ones with the best prompts on day one.
They are the ones that convert each cycle into better systems.
